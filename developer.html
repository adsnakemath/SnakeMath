About the App

SnakeMath is a skill-based mobile application that combines classic Snake gameplay with math challenges, competitive rankings, and a Solana-powered reward system. Players solve math problems to grow their snake, earn XP, level up, and compete for seasonal SKR token pools.

Key Features

  • 🎮 Snake gameplay with real-time math questions
  • 🏆 Global leaderboard with seasonal reward pools
  • ⭐ XP system with levels and achievements
  • 💰 Deposit and withdraw SKR tokens via Solana blockchain
  • 🎁 Purchase extra game turns with SKR tokens
  • 📱 Progressive Web App (PWA) — works on any device

How Rewards Work

Each season, SKR tokens accumulate in a reward pool. At the end of the season, the pool is distributed to top-ranked players based on their scores. Players can also use SKR tokens to buy additional game turns and send donations to other users.
